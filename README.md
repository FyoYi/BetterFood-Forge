## English

This is a **Forge** mod for **Minecraft 1.20.1**.

###  Key Features

This mod aims to provide a high-performance and immersive food system:

*   **Lag-free Decay**: Uses a timestamp mechanism to calculate decay, completely solving the "hand shaking" animation and server lag issues found in traditional mods. Supports decay in inventories, chests, and dropped items.
*   **Smart Stacking**: Optimized interaction logic. Left-click retains vanilla swap behavior, while **Right-click** intelligently merges items of different freshness and calculates the average shelf life.
*   **Visual UI**: Displays freshness via the durability bar and detailed countdowns in the item tooltip.
*   **Highly Configurable**: Supports Data Pack configuration and provides an in-game menu (`/betterfood menu`) to quickly adjust shelf life for any food.

## 简体中文

这是一个适用于 **Minecraft 1.20.1** 的 **Forge** 模组。

###  主要功能

本模组旨在提供一个高性能且沉浸的食物系统：

*   **无卡顿腐烂**：采用时间戳机制计算腐烂，彻底解决传统模组导致的“手持物品抖动”和服务器卡顿问题。支持背包、箱子及掉落物全场景腐烂。
*   **智能堆叠**：优化了交互逻辑，左键保持原版交换，**右键**同类食物可智能合并并计算平均保质期。
*   **可视化 UI**：通过耐久条和悬停提示（Tooltip）实时显示食物的新鲜度与精确倒计时。
*   **高度自由**：支持数据包配置，并提供游戏内菜单 (`/betterfood menu`) 快速调整任意食物的保质期。
